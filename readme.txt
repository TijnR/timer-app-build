Dit is een van mijn eerste projecten gemaakt met Typescript en React.
Verder is dit ook de eerste keer dat het mij gelukt is om er een PWA van te maken.

Ik kwam op dit project omdat ik online opzoek was naar een timer voor werk intervals, maar alle site waren een beetje outdated.
Dus ik dacht dat dit wel een mooie challenge kon zijn. 
Ik wilde de timer maken zonder enige begeleiding. 
Dit zorgde wel voor wat uitdagingen, maar heb nu wel meer geleerd.
Het moeilijkste vond ik om een setInterval te bebruiken in combinatie met state.

Ik het dit projectje in een week gemaakt en ben blij met het resultaat. 
Ook heb ik nieuwe dingen geleerd over PWA's en react.
